# Support

Use this page for MyTaxReturn AU support, beta feedback, and app review contact details.

## Before Sending Feedback

Do not include tax file numbers, identity documents, bank details, myGov credentials, employer payroll identifiers, or screenshots containing sensitive personal information.

The app is an estimate calculator only. It is not affiliated with the Australian Taxation Office and is not operated by a registered tax agent.

## Calculation Issue Template

When reporting a calculation concern, include:

- App version and build number.
- Platform and device model.
- Selected financial year.
- Which step looked wrong: income, deductions, Medicare, HELP, PAYG withholding, private health, spouse/family, results, save, export, or scenario comparison.
- A rounded or anonymized example of the input category that produced the issue.
- The result category that looked wrong.
- Whether the issue happens after restarting the app.
- Any official ATO page or notice you believe conflicts with the estimate.

Do not include exact sensitive tax records if rounded or anonymized numbers are enough to reproduce the problem.

## Beta Feedback

Beta users can report structured feedback from the app:

About -> Beta Feedback

Direct template URL:

https://github.com/aanujkhurana/TaxReturnCalculator/issues/new?template=beta-feedback.md

## Bug Report Template

When reporting an app bug, include:

- App version and build number.
- Platform and device model.
- Steps to reproduce.
- Expected result.
- Actual result.
- Whether the issue happens in light mode, dark mode, or both.
- Whether the issue affects saving, exporting, or deleting local data.

## Store Review Contact

For App Store or Google Play review, use the repository issue tracker or configured support URL in the store listing.

Production releases should replace this repository page with a stable branded support page or support mailbox before broad launch.
